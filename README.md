# 🚀 StockPilot 

AI-powered stock market analysis platform built with FastAPI + Streamlit.

## Features

- 📊 Candlestick charts
- 📈 Technical indicators (MA, RSI)
- 🤖 ML prediction
- 🧠 LSTM model
- 📉 Risk analysis
- 📊 Backtesting
- 📁 Portfolio tracking
- 🔔 Alerts

## Tech Stack

- FastAPI
- Streamlit
- yFinance
- Scikit-learn
- TensorFlow
- Plotly
